# DNS

* DNS is used to convert human friendly domain names into ip address
* Two different types of IP address:
    * IPV4
    * IPV6
* Top level Domains:
    * .com
    * .edu
    * .gov
* second level Domains:
    * .co.uk
    * .gov.uk
    * .com.au
* Domain Registrars
* SOA Records
* NS Records
* A Records
* TTL Records
* CNames
* Alias Records

## Routing policies

* simple
* weightage
* latency
* failover
* geolocation
